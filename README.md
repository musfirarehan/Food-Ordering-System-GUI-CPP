# 🍔 Online Food Ordering System (C++ & Raylib)

A desktop-based Online Food Ordering System developed in C++ using Object-Oriented Programming principles and a Raylib-powered graphical user interface.

## 📖 Overview

This project was developed as a second-semester Object-Oriented Programming project at FAST-NUCES. The system enables customers to:

* Register and log into accounts
* Browse categorized food menus
* Add and remove items from a shopping cart
* Apply promotional discount codes
* Place orders and view receipts
* Review order history
* Persist data across sessions through file handling

## ✨ Features

### User Management

* Registration and Login
* Duplicate username prevention
* Persistent customer data

### Food Ordering

* Categorized menu
* Shopping cart management
* Quantity merging

### Discounts

* Promotional code system
* Dynamic price calculations

* Scrollable order history

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/musfirarehan/Food-Ordering-System-GUI-CPP.git
```

2. Open the project in Visual Studio Code.

3. Install Raylib.

4. Build and run the project.
